# Yusra Quantum AI Assets

This directory contains static assets for the Yusra AI application.

## Structure
- /logo.svg: Vector logo for high-resolution displays.
- /logo.png: Raster logo placeholder.
- /favicon.ico: Browser tab icon placeholder.
- /media/: Directory for video and multimedia assets.

## Media Support
Supported formats include:
- MP4 (H.264/AAC recommended for web)
- MPEG
- AVI

Note: The binary files (.png, .ico, .mp4, etc.) in this folder are text placeholders. Please replace them with actual binary media files before production deployment.
